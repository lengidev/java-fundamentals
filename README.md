# Java Coursework

This repository contains Java exercises and small programs created as part of my Computer Science coursework.

The goal of this repository is to build a strong foundation in Java by practicing core concepts and applying them through simple, well-structured programs.

## Structure
- `src/` – Java source files organized by week or topic  
- `notes/` – Personal notes and explanations of key concepts

## Topics Covered
- Java basics
- Variables and data types
- Control flow
- Methods and classes
- Object-oriented programming fundamentals
